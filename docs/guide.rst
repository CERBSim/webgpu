User Guide
==========

Work through these notebooks in order, or jump to the topic you need.
Each notebook is self-contained and can be run locally in Jupyter.

The guide starts with the low-level WebGPU basics (shaders, buffers,
pipelines) and then introduces the convenience renderers built for
scientific visualization.
